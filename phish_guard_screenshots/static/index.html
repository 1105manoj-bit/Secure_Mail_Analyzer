About PhishGuard
// Real-world email security analysis platform
🔐

Authentication Verification

Live DNS checks for SPF, DKIM, and DMARC. Detects spoofing, missing policies, and misconfigured records with letter grades.

🎣

Phishing Detection

Heuristic engine with 100+ indicators: urgency patterns, threat language, credential harvesting, BEC fraud, and brand impersonation.

🌐

URL Reputation

Analyzes URLs for IP-hosting, URL shorteners, homograph attacks, deceptive paths, free TLDs, and SSL validity.

🔍

IOC Extraction

Automatically extracts MD5/SHA1/SHA256 hashes, IPv4 addresses, email addresses, and Bitcoin wallet addresses from any text.

📎

Attachment Analysis

Detects dangerous file types (.exe, .js, .vbs), double extension tricks, misleading extensions, and macro-enabled Office files.

🗺

Routing Forensics

Traces email routing hops from Received headers, extracts sending IPs, and identifies geographic anomalies.
